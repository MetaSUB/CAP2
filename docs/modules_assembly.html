Assembly¶

The assembly subpipeline generates large sequence assemblies and analyzes them for a number of analyses. It runs on data that has been processed with the preprocessing pipeline.

Modules¶

MetaSPAdes¶

MetaSPAdes generates high quality assemblies of microbial genomes from metagenomic data. The assemblies can be used for a number of downstream analyses including identifying Metagenome Assembled Genomes, gene clusters, and refined taxonomic profiles.

MetaSPAdes produces a number of output files which are detailed here. Most downstream analyses use eitheer the contigs or scaffolds file. An example of the output files from this module may be found on Pangea.
